# RWB Aura Migraine Simulator Mobile

RWB Aura Migraine Simulator Mobile is an experimental Python application that simulates visual migraine aura patterns.

The program was created as part of the **RWB Tech Lab** experiments to explore how migraine aura visuals may appear when generated through RGB image sequences.

This version is designed to run on mobile devices using Python environments such as Pydroid and uses Pygame to display the visual simulation.

## Purpose

Migraine aura often appears as moving visual patterns, zig-zag shapes or flickering light effects.  
The goal of this experiment is to simulate similar visual patterns to better understand how such effects may appear on a screen.

This tool is intended for experimentation, visualization and discussion of migraine aura phenomena.

It is **not a medical device** and should not be used for diagnosis or medical purposes.

## Features

- Visual aura simulation using RGB image sequences
- Mobile friendly interface
- Experimental graphical rendering using Python
- Designed for testing visual timing and color effects
